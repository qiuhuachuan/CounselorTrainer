# AI来访者培训系统 - 前端

这是一个基于Vue.js的AI来访者培训系统前端应用，专为心理咨询师培训设计。

## 功能特性

### 🎯 核心功能
- **用户注册与登录**：完整的用户认证系统
- **个人仪表板**：用户个人界面，包含对话历史管理
- **AI来访者对话**：与不同类型AI来访者的模拟对话练习
- **实时计时器**：对话过程中的时间记录
- **轮次追踪**：清晰显示当前对话轮数
- **流式输出**：AI回复的流式输出效果

### 🎨 界面特色
- **响应式设计**：适配不同屏幕尺寸
- **现代化UI**：美观的用户界面设计
- **折叠式侧边栏**：可折叠的对话历史归档
- **智能提示框**：提供专业的回复建议
- **实时状态指示**：清晰显示对话状态

### 🔧 技术实现
- **Vue 3**：使用最新的Vue.js框架
- **TypeScript**：类型安全的开发体验
- **Pinia**：现代化的状态管理
- **Vue Router**：单页面应用路由管理
- **流式输出**：模拟真实的对话体验

## 界面布局

### 登录/注册界面
- 简洁的表单设计
- 表单验证和错误提示
- 注册后自动跳转到登录界面

### 主界面（Dashboard）
- **顶部导航栏**：显示用户名和退出登录
- **左侧折叠栏**：
  - 新建对话按钮
  - 对话历史归档
  - 可折叠/展开功能
- **主内容区域**：
  - 欢迎界面（无对话时）
  - 来访者选择弹窗

### 对话界面
- **顶部工具栏**：
  - 会话信息显示
  - 当前轮次计数器
  - 实时计时器
- **对话消息区域**：
  - 用户和AI消息区分显示
  - 轮次标识
  - 时间戳显示
  - 流式输出效果
- **右侧提示框**：
  - 专业回复建议
  - 可刷新获取新建议
- **底部输入区域**：
  - 智能输入框（自动调整高度）
  - 发送按钮（输入内容后启用）
  - 输入状态管理

## 来访者类型

系统提供四种类型的AI来访者供选择练习：

1. **焦虑型来访者** 😰 - 表现出焦虑症状的来访者
2. **抑郁型来访者** 😔 - 情绪低落、兴趣减退的来访者  
3. **愤怒型来访者** 😠 - 情绪激动、易怒的来访者
4. **依赖型来访者** 🥺 - 过度依赖、缺乏自信的来访者

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 使用流程

1. **注册账号**：首次使用需要注册新账号
2. **登录系统**：使用注册的账号登录
3. **选择来访者**：点击"新建对话"选择要练习的来访者类型
4. **开始对话**：与AI来访者进行对话练习
5. **查看历史**：在左侧边栏查看历史对话记录
6. **获取提示**：参考右侧提示框的专业建议

## 技术架构

```
src/
├── assets/          # 静态资源
├── components/      # 可复用组件
├── router/          # 路由配置
├── stores/          # Pinia状态管理
├── views/           # 页面组件
├── App.vue          # 根组件
└── main.ts          # 应用入口
```

## 状态管理

使用Pinia进行状态管理，包含两个主要store：

- **auth**：用户认证状态管理
- **chat**：对话状态管理（会话、消息、计时器等）

## 开发说明

### 路由配置
- `/login` - 登录页面
- `/register` - 注册页面  
- `/dashboard` - 主界面（需要认证）
- `/chat/:sessionId?` - 对话界面（需要认证）
